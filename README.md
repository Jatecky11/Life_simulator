# Life_simulator
# Консольный симулятор жизни

**Описание:** простой симулятор экосистемы с взаимодействием популяций (хищники и жертвы).

**Используемые классы:**
- `Organism` — базовый организм с энергией и именем;
- `Population` — группа организмов одного вида;
- `Ecosystem` — среда взаимодействия популяций.

**Как запустить:**
1. Клонируйте репозиторий.
2. Запустите `python main.py`.

**Особенности:**
- Организмы теряют энергию каждый день.
- Хищники (лисы) могут есть жертв (зайцев).
- Система выводит статус экосистемы после каждого дня.

**Требования:**
- Python 3.x
